🧪 Offerio Embed Test

ℹ️ Info: Diese Seite testet verschiedene Methoden zur Einbettung des Offerio-Formulars.

Methode 1 Direkter iFrame

Die einfachste Methode - fügen Sie einfach einen iFrame ein:

<iframe src="https://offerio.ch/embed/default" width="100%" height="800" frameborder="0" style="border: none;"></iframe>

Methode 2 JavaScript Widget

Mit dem JavaScript-Widget und automatischer Höhenanpassung:

<script src="https://offerio.ch/embed.js"></script> <div id="my-form"></div> <script> Offerio.embed('my-form', { formSlug: 'default', lang: 'de', height: 'auto' }); </script>

Methode 3 Data Attributes (Auto-Init)

Automatische Initialisierung mit data-Attributen:

<script src="https://offerio.ch/embed.js"></script> <div data-offerio-embed="default" data-lang="de" data-height="auto"></div>

📋 Verfügbare Parameter

Parameter Typ Beschreibung
formSlug string Formular-Slug aus dem Admin-Bereich
category string Vorausgewählte Kategorie (umzug, reinigung, etc.)
lang string Sprache: de, en, fr, it
color string Primärfarbe als Hex (ohne #)
hideHeader boolean Header ausblenden
height string 'auto' oder feste Höhe wie '600px'
🔒 Sicherheitshinweis: Die Einbettung ist auf Domains mit .ch, .com, .de und .at sowie localhost beschränkt.
